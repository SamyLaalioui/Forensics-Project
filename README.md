# USB Device Forensics Analyzer

## Project Overview
Analyzes Windows USB device history to identify connected devices, timestamps, and potential security risks.

## Evidence
Simulated USB connection data using Python. Includes normal devices and suspicious patterns.

## Tools
- Python 3.8+
- Pandas for analysis
- CSV for data storage

## Results
Found 2 suspicious USB devices with unusual connection patterns.
